<strong>80604 Change posting date when posting sales invoice</strong>
<ul><li>The ability to set the Posting Date, Document Date, and VAT Date when creating a sales invoice from Hotel Invoice Management was added.</li><li>A new dialog now allows the user to review and modify these dates before the invoice is created.</li><li>All dates default to the current work date, preserving the previous behavior when no changes are made.</li></ul>
