<strong>80899 Voyado Enter Press and Search Fixes</strong>
<ul><li>There was an issue where pressing <b>Enter</b> after entering search criteria (Such as, phone number, email, or SSN) would close the search window without executing the search. The Enter key now correctly triggers the member lookup.</li><li>The SSN input dialog was updated to use a numeric keypad interface.</li></ul>
