<strong>80521 AE - It is possibe to add an ePre to existing AE order</strong>
<ul><li>Removing unnecessary options, now only the actions for AE orders are available.<ul><li>Actions removed:<ul><li>New e-prescription</li><li>Mark item as retail (Yes / No)</li><li>Earlier withdrawals</li><li>Auto-split an order</li><li>Split an order</li><li>Split a line</li><li>Create a sibling e‑Prescription</li></ul></li></ul></li></ul>
<strong>80524 AE - Sync Prescription Order and Order Line to Posted</strong>
<ul><li>When viewing a <b>Posted Prescription Order</b> for an Assistive Equipment order, the order lines now display the correct item information.</li><li><b>Non-assistive equipment orders</b> are unaffected and continue to display the same information as before.</li></ul>
<strong>81767 Wrong Dispense Order Lines updated when using "Post & Create Invoices""</strong>
<ul><li>Dispense Order lines are correctly updated after using <b>Post &amp; Create Invoices</b> action in <b>Dispense Production</b> Order List.</li></ul>
<strong>81910 Wrong values in "Remaining Quantity (mg)", "Dispensed Quantity (mg)" and "Prescribed Total Quantity (mg)" of Posted Production Orders</strong>
<ul><li>There was an issue with <b>Remaining Quantity (mg)</b>, <b>Dispensed Quantity (mg)</b> and <b>Prescribed Total Quantity (mg)</b> of Posted Production Orders and incorrect values. This was fixed. </li></ul>
